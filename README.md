# Pulih — Recovery Tracker

Aplikasi pemulihan pribadi berbasis PWA. Offline, privat, semua data tersimpan di perangkat.

## Fitur
- Streak tracker harian dengan milestone
- Jurnal suasana hati + grafik mood 30 hari
- Tameng (toolkit dorongan) dengan breathing 4-7-8 dan timer 20 menit
- Target pemulihan 5 fase (35 item checklist)
- Relapse tracker dengan pelajaran
- Alasan berhenti + popup darurat
- Afirmasi harian
- Zona merah (jam berisiko otomatis memperingatkan)
- Streak Freeze (2x/bulan)
- PIN Lock privasi
- Notifikasi harian
- Dark mode

## Cara install di Android
1. Buka di Chrome: https://[username].github.io/pulih-app
2. Tap menu ⋮ → "Tambahkan ke layar utama"
3. Selesai — ikon Pulih muncul di home screen

## Teknologi
- Pure HTML/CSS/JS — tidak ada framework
- PWA dengan Service Worker (offline penuh)
- localStorage untuk penyimpanan data lokal

## Privasi
Tidak ada server, tidak ada analytics, tidak ada data yang dikirim keluar. 100% lokal.
